File Transfer Protocol Secure (FTPS) is a service Deltares employees seeking a reliable and secure data transfer solution. It employs the standard FTP protocol that has added strong encryption methods to safeguard your data from unauthorized access and tampering during transit. FTPS also offers authentication mechanisms to ensure that only authorized users can access and transfer files. 

## Short description 

A secure data transfer solution for Deltares employees. 

## Cost 

None (incorporated into ICT costs) 

## Max. storage size 

50 Gb (on request, default size unknown) 

## Sharing and collaboration 

Deltares FTPS offers Deltares employees an easy way to share and synchronize files in a secure manner.
